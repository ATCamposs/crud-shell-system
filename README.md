# Crud shell system

## Como iniciar o projeto
 - Clone o reposit√≥rio
   - `git clone git@github.com:ATCamposs/crud-shell-system.git`
 - Entre na pasta principal
   - `cd crud-shell-system`
 - Copie o arquivo .env.backup para .env
   - `cp .env.backup .env`
 - Instale os containers (aguarde)
   - `docker-compose up -d`
 - Entre pelo link em seu navegador
   - `http://localhost:8080`
